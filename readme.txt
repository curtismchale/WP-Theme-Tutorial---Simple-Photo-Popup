== Description ==

Adds a custom post type to your WordPress site to display photos.

Photos can be displayed on any post or page by using the
[thwp_photo_gallery] shortcode.

The size of the image that shows in the popup is controlled by the
medium images size found under Settings/Media

See screenshots in the plugin for an idea of how it will look.

== Installation ==

1. Extract to your wp-content/plugins/ folder.

2. Activate the plugin.

3. Add photos by adding items to the coupon custom post type.

4. Add [thwp_photo_gallery] to a post or page to show the photos.
